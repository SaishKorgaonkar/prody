# Prody Marketing Landing

Launch-ready marketing site for **Prody**, your autonomous engineering team.

## Run

```bash
cd landing
npm install
npm run dev
```

Open http://localhost:3000

## Page flow

1. **Hero**: Prody wordmark, tagline, stats, looping product video
2. **Trust strip**: audience types
3. **Problem** → **Use cases** → **Two surfaces** → **How it works**
4. **Agents** → **Capabilities** → **Readiness score** → **Why Prody**
5. **CTA** → **Footer**

## Hero video

`public/hero-loop.mp4` plays in the glass panel on the hero right column.
